# CEAClust for Textual Data

A Jupyter Notebook project for clustering textual data using CEAClust algorithm.

## Technologies
- **Language**: Jupyter Notebook (Python)

## Purpose
Implement and explore the CEAClust clustering algorithm for text data analysis.

## Features
- Text data clustering
- Data visualization
- Algorithm implementation
- Educational notebooks

## Usage
Open the Jupyter notebooks to explore CEAClust clustering on textual datasets.
